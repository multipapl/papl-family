# Family Canvas

Интерактивное семейное дерево для семьи.

## Главный приоритет

Этот проект делается в первую очередь не для “продвинутых пользователей” и не для десктопа.

Он делается для:

- пожилых родственников;
- обычных членов семьи;
- использования с телефона или планшета;
- максимально простого и спокойного взаимодействия.

Если в будущем какая-то идея будет красивой, но сложной, приоритет всегда такой:

1. понятно пожилому человеку;
2. удобно с телефона;
3. только потом эффектно и технологично.

## Что это за продукт

Не “редактор графа”, а семейное приложение с картой родства.

Правильный сценарий использования:

1. найти человека;
2. открыть его карточку;
3. нажать одну из простых кнопок: `Моя ветка`, `Предки`, `Потомки`, `Все дерево`;
4. при необходимости перейти к карте;
5. редактирование открывать отдельно.

## Текущее состояние

Сейчас в репозитории лежит локальный mobile-first прототип без базы данных.

Уже есть:

- поиск;
- карточка человека;
- крупные действия для навигации;
- карта семьи;
- локальный режим редактирования;
- документ с полным направлением проекта.
